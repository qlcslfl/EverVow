# Wedding Invite

Next.js + TailwindCSS + Supabase + Spring Boot 기반 청첩장 서비스

이 프로젝트는 모바일 청첩장 제작을 위한 서비스입니다. 사용자는 정보를 입력하고, 다양한 디자인을 선택하여 나만의 청첩장을 만들 수 있습니다. 샘플 미리보기와 최종본 링크 제공 기능도 지원합니다.

## 주요 기능
- 고객 정보 입력 폼
- 여러 디자인 리스트업 및 선택
- 꾸미기(스티커 등) 기능
- 샘플 제작 및 미리보기
- 최종본 링크 공유

## 폴더 구조
- frontend: Next.js + TailwindCSS
- backend: Spring Boot 또는 Supabase API
- database: DB 스키마 및 마이그레이션
- storage: 이미지/꾸미기 요소 저장
- docs: 프로젝트 문서

## 실행 방법
1. frontend 폴더에서 `npm install` 후 `npm run dev`로 개발 서버 실행
2. backend 폴더에서 Spring Boot 실행

자세한 내용은 각 폴더의 README.md를 참고하세요.
